# Gematria Chrome Extension

Unpacked extension source for the standalone gematria module.

## Load locally

1. Open `chrome://extensions`.
2. Enable **Developer mode**.
3. Click **Load unpacked**.
4. Select this directory: `gematria/plugin`.

## Features

- Twitter/X tweet hover value cards for selected cyphers
- Interesting-value tweet highlighting
- Composer values near the audience selector
- Right-click context menu for selected text on non-Twitter pages
- Local save action for phrase/value/link entries
- Saved list renderer on `/gematria/saved`
